# JWT Mobile Client

Мобильное приложение для регистрации и авторизации с JWT-токеном.

## Функционал
- Регистрация пользователя
- Авторизация с получением JWT токена
- Запрос к защищенному API
- Логирование всех действий
- Отображение сервисных сообщений

## Запуск
1. Запустите сервер из Лабораторной работы №1
2. Соберите проект в Android Studio
3. Установите приложение на эмулятор или устройство

## Настройка подключения
По умолчанию сервер: `http://10.0.2.2:8080/` (для эмулятора Android)
Для реального устройства измените IP в `ApiClient.kt`
